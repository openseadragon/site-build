example: keyboard navigation

The viewer supports keyboard operations to complement the drag and scroll (zoom) mouse interactions.

To interact via the keyboard, the focus must be on the viewer. In this example the focus is set using document.getElementById('contentDiv').focus();

You can use the following keys to navigate:

  • [ w, up arrow ] - move viewport up
  • [ s, down arrow ] - move viewport down
  • [ a, left arrow ] - move viewport left
  • [ f, right arrow ] - move viewport right
  • [ 0 ] - zoom / move viewport home
  • [ - / _, shift+W, shift+up arrow ] - zoom viewport out
  • [ = / +, shift+S, shift+down arrow ] - zoom viewport in

Note that Chrome and Safari do not support the arrow keys for these operations.
