example: image collections

The image collection is a major hurdle for OpenSeadragon. It allows arbitrarily large sets of tile sources, of mixed types, to be used to easily visualize the group as a "wall" or "grid". Many core features are available for preview now, though the functionality is not yet complete.

When complete, this feature will make OpenSeadragon into a whole new platform for zoomable image visualization.

Basic Single-Row Tile Source Collection

One Row, Horizontal 
OpenSeadragon({
    ...
    collectionMode:       true,
    collectionRows:       1, 
    collectionTileSize:   1024,
    collectionTileMargin: 256,
    ...
});
