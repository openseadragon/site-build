Installing

OpenSeadragon has no external dependencies. To install, just copy the JavaScript and images into your web root or static files directory. You can examine the source of this page to see how to include OpenSeadragon in a page.

Development

OpenSeadragon is developed on GitHub. To report an issue, use our issue tracker. You can clone the source repository with Git by running: 

git clone git://github.com/openseadragon/openseadragon
